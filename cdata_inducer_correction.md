# Поправка к модели CDATA: Индукторы дифференцировки на центриолях

## Суть поправки

Текущая реализация (`CentriolarInducers`) хранит единый счётчик `s_count` для всей клетки.
По теории CDATA индукторы привязаны **к каждой центриоли отдельно** и имеют разный состав
у материнской и дочерней центриоли. Асимметрия дифференцировки возникает не из возраста,
а из **того, на каком комплекте ещё остались индукторы** в момент, когда O₂ достигает центра.

---

## Расположение индукторов

```
[ Материнская центриоль ]          [ Дочерняя центриоль ]
  ┌─────────────────────┐            ┌─────────────────────┐
  │  Комплект M         │            │  Комплект D         │
  │  [I₁][I₂][I₃]...   │            │  [J₁][J₂][J₃]...   │
  └─────────────────────┘            └─────────────────────┘
        ↑ свой тип индукторов              ↑ другой тип
```

- Комплект M и Комплект D — **разные молекулы**, запускают разные программы специализации
- Оба комплекта нужны для полноценной тотипотентности
- Каждый комплект убывает независимо

---

## Механизм отделения индукторов при O₂-воздействии

### Случай 1: Тотипотент / Плюрипотент — оба комплекта ещё есть

O₂ достигает центра → отщепляет **сразу от обоих** комплектов:

```
O₂ достигает центриолярной зоны
         │
         ├─► M-комплект: отщепляется 1 индуктор (вероятность Pₘ)
         │
         └─► D-комплект: отщепляется 1 индуктор (вероятность P_d)

Каждый отщеплённый индуктор уходит в цитоплазму и активирует
свою дифференцировочную программу (разные сигналы от M и D).
```

После этого события клетка становится чуть менее потентной.
При следующем делении **обе дочерние клетки** наследуют уменьшенные комплекты —
одна (от M) с `Pₘ_count - 1`, другая (от D) с `P_d_count - 1`.

### Случай 2: На одной центриоли индукторы уже закончились

Тогда O₂ может отщепить **только от той, у которой они ещё есть**:

```
M = 0,  D ≥ 1:
         │
         └─► D-комплект: отщепляется 1 индуктор (вероятность P_d)
             M-комплект: пустой, ничего не происходит

M ≥ 1,  D = 0:
         │
         └─► M-комплект: отщепляется 1 индуктор (вероятность Pₘ)
             D-комплект: пустой, ничего не происходит
```

Когда последний индуктор отщепляется с последнего непустого комплекта → `M=0, D=0`
→ **запускается путь запрограммированного апоптоза**.

## Уровни потентности

```
Состояние комплектов                               Потентность
──────────────────────────────────────────────────────────────
M = полный  И  D = полный                      →  Тотипотент
M ≥ 1  И  D ≥ 1  (не оба полные)              →  Плюрипотент
(M = 0 И D ≥ 2) ИЛИ (D = 0 И M ≥ 2)          →  Олигопотент
(M = 0 И D = 1) ИЛИ (D = 0 И M = 1)          →  Унипотент
M = 0  И  D = 0                               →  Апоптоз
```

**Таблица переходов:**

```
            D: полный     D: частичный   D: 1         D: 0
          ┌─────────────┬──────────────┬────────────┬───────────┐
M: полный │ ТОТИПОТЕНТ  │ ПЛЮРИПОТЕНТ  │ ОЛИГОПОТЕНТ│ ОЛИГОПОТ. │
M: частич.│ ПЛЮРИПОТЕНТ │ ПЛЮРИПОТЕНТ  │ ОЛИГОПОТЕНТ│ ОЛИГОПОТ. │
M: 1      │ ОЛИГОПОТЕНТ │ ОЛИГОПОТЕНТ  │ ПЛЮРИПОТЕНТ│ УНИПОТЕНТ │
M: 0      │ ОЛИГОПОТЕНТ │ ОЛИГОПОТЕНТ  │ УНИПОТЕНТ  │ АПОПТОЗ   │
          └─────────────┴──────────────┴────────────┴───────────┘
```

## Вероятности отщепления: Pₘ и P_d

Вероятности отщепления от материнской и дочерней центриоли **в общем случае не равны**.
Это вводится как **настраиваемый параметр** — процентное соотношение.

```rust
pub struct InducerDetachmentParams {
    /// Базовая вероятность отщепления при данном уровне O₂ (у обоих)
    pub base_detach_probability: f32,

    /// Доля вероятности, приходящаяся на материнскую центриоль [0.0 .. 1.0]
    /// 0.5 = равновероятно
    /// > 0.5 = материнская теряет индукторы чаще
    /// < 0.5 = дочерняя теряет чаще
    pub mother_bias: f32,

    /// Возраст клетки (лет) — вспомогательный параметр, влияет на mother_bias
    /// (с возрастом материнская центриоль накапливает PTM → слабее удерживает индукторы)
    /// Сам по себе возраст НЕ является причиной отщепления.
    pub age_years: f32,

    /// Коэффициент влияния возраста на mother_bias
    /// effective_mother_bias = mother_bias + age_years * age_bias_coefficient
    pub age_bias_coefficient: f32,
}

impl Default for InducerDetachmentParams {
    fn default() -> Self {
        Self {
            base_detach_probability: 0.1,
            mother_bias: 0.6,       // материнская чуть уязвимее по умолчанию
            age_years: 0.0,
            age_bias_coefficient: 0.003, // +0.3% на год → к 40 годам +12%
        }
    }
}

impl InducerDetachmentParams {
    /// Фактическое соотношение с учётом возраста
    pub fn effective_mother_bias(&self) -> f32 {
        (self.mother_bias + self.age_years * self.age_bias_coefficient).min(0.95)
    }

    /// Вероятность отщепления от материнской при данном oxygen_level
    pub fn mother_prob(&self, oxygen_level: f32) -> f32 {
        oxygen_level * self.base_detach_probability * self.effective_mother_bias()
    }

    /// Вероятность отщепления от дочерней при данном oxygen_level
    pub fn daughter_prob(&self, oxygen_level: f32) -> f32 {
        oxygen_level * self.base_detach_probability * (1.0 - self.effective_mother_bias())
    }
}
```

## Алгоритм отщепления (логика step())

```
fn detach_by_oxygen(pair: &mut CentriolarInducerPair,
                    oxygen_level: f32,
                    params: &InducerDetachmentParams):

    let m_has = pair.mother_set.remaining > 0
    let d_has = pair.daughter_set.remaining > 0

    match (m_has, d_has):

        (true, true):
            // Случай 1: оба комплекта непустые → отщепляем от обоих
            if rand() < params.mother_prob(oxygen_level):
                pair.mother_set.detach_one()
            if rand() < params.daughter_prob(oxygen_level):
                pair.daughter_set.detach_one()

        (false, true):
            // Случай 2a: M пуст → отщепляем только от D
            // вероятность = полная базовая (mother_bias уже не применяется)
            if rand() < oxygen_level * params.base_detach_probability:
                pair.daughter_set.detach_one()

        (true, false):
            // Случай 2b: D пуст → отщепляем только от M
            if rand() < oxygen_level * params.base_detach_probability:
                pair.mother_set.detach_one()

        (false, false):
            // Оба пусты → апоптоз уже должен быть запущен ранее
            trigger_apoptosis(entity)

    // После отщепления пересчитать потентность
    pair.update_potency()
```

## Правило синтеза при делении (наследование)

При синтезе новой дочерней центриоли на ней синтезируются индукторы
**в том количестве, которое есть на родительской прямо сейчас**, — не в исходном максимуме.

```
Состояние до деления:
  M = 6 (изначально было 10, 4 уже отщепились)
  D = 3 (изначально было 8, 5 уже отщепились)

После деления:
  Клетка A (наследует материнскую):
    ┌ новая M-центриоль: старая (M=6, без изменений)
    └ новая D-центриоль: синтезируется с 6 ← копирует ТЕКУЩИЙ остаток M

  Клетка B (наследует дочернюю):
    ┌ новая M-центриоль: старая дочерняя (D=3, теперь стала материнской)
    └ новая D-центриоль: синтезируется с 3 ← копирует ТЕКУЩИЙ остаток D
```

Если индуктор ранее отщепился — он **не восстанавливается** в новом комплекте.
Потеря наследуется всеми потомками.

## Сводная схема всей логики

```
         O₂ ДОСТИГАЕТ ЦЕНТРИОЛЕЙ
                  │
         ┌────────┴────────┐
         ▼                 ▼
    M пуст?           D пуст?
         │                 │
   нет → отщепить    нет → отщепить
   да  → пропустить  да  → пропустить
         │                 │
         └────────┬────────┘
                  │
          Пересчёт потентности:
          ┌────────────────────────────────┐
          │ M=full & D=full → Тотипотент  │
          │ M≥1  & D≥1     → Плюрипотент │
          │ M=0  & D≥2     → Олигопотент │
          │ M=0  & D=1     → Унипотент   │
          │ M=0  & D=0     → АПОПТОЗ     │
          └────────────────────────────────┘
                  │
         Деление → наследование текущих остатков
```

## Параметры для панели управления

| Параметр | Тип | Диапазон | По умолчанию | Описание |
|----------|-----|----------|--------------|----------|
| `mother_bias` | f32 | 0.0 – 1.0 | 0.6 | Доля вероятности отщепления от материнской центриоли |
| `base_detach_probability` | f32 | 0.0 – 1.0 | 0.1 | Базовая вероятность при данном уровне O₂ |
| `age_bias_coefficient` | f32 | 0.0 – 0.02 | 0.003 | Насколько возраст сдвигает mother_bias (дополнительный параметр) |
| `mother_inducer_count` | u32 | 1 – 100 | 10 | Исходный размер M-комплекта |
| `daughter_inducer_count` | u32 | 1 – 100 | 8 | Исходный размер D-комплекта |

**Важно:** возраст `age_years` — вспомогательный параметр, лишь сдвигающий `mother_bias`.
Он **не является причиной** отщепления — причиной всегда является O₂.

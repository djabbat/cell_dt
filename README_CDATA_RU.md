## 1. Суть гипотезы CDATA

**CDATA (Centriolar Damage Accumulation Theory of Aging)** — теория, объясняющая
старение организма как неизбежное следствие реализации программы дифференцировки
клеток.

Центральный тезис: **материнская центриоль стволовой клетки является
единственной биологической структурой, которая накапливает необратимые
молекулярные повреждения на протяжении всей жизни организма**, потому что:

1. Центриоль реплицируется по матричному принципу — старая (материнская) никогда
   не восстанавливается полностью заново.
2. При асимметричном делении стволовой клетки дочь, сохраняющая стволовость,
   **всегда удерживает старую материнскую центриоль**.
3. Это означает, что вся хронологически накопленная молекулярная «порча»
   остаётся в стволовой клетке — тогда как заменённые дифференцированные
   клетки получают относительно «новые» центриоли.

Парадокс старения: ткани непрерывно обновляются стволовыми клетками, но
организм всё равно стареет — именно потому, что стволовые клетки несут
в своих материнских центриолях нарастающий груз повреждений.

## 2. Ключевые механизмы

### 2.1 Система индукторов дифференцировки

По теории (Tkemaladze 2005/2023), центриоль содержит два набора
**индукторов необратимых  дифференциаций**:

| Структура | Назначение | Начальный запас |
|-----------|------------|-----------------|
| **S-структура** | Соматическая линия | ~50 индукторов (≈ лимит Хейфлика) |
| **H-структура** | Гаметная линия | ~? индуктора (до мейоза) |

При каждом **дифференцирующем митозе** один S-индуктор высвобождается из
центриоли, входит в ядро, выключает одну генную сеть и включает другую —
это и есть необратимое изменение морфогенетического статуса.

```
S_count = 50  →  тотипотентная (зигота)
S_count = 40  →  плюрипотентная (ВКМ бластоцисты)
S_count = 25  →  мультипотентная (взрослые СК)
S_count = 10  →  унипотентная
S_count = 0   →  терминальная дифференцировка / апоптоз
```

### 2.2 Накопление молекулярных повреждений

Пять типов необратимых повреждений накапливаются в материнской центриоли:

```
СТРЕССОРЫ (ROS, УФ, метаболиты)
        │
        ▼
МАТЕРИНСКАЯ ЦЕНТРИОЛЬ — необновляемый хранитель времени
        │
        ├── Карбонилирование белков (SAS-6, CEP135)
        │     → разрушение структурной целостности, дестабилизация PCM
        │
        ├── Гиперацетилирование α-тубулина (↓ HDAC6, ↓ SIRT2)
        │     → снижение гибкости микротрубочек, нарушение транспорта
        │
        ├── Агрегаты белков (CPAP, CEP290)
        │     → блок аппарата дупликации, дезорганизация PCM
        │
        ├── Дисбаланс фосфорилирования (PLK4, NEK2, PP1)
        │     → эктопическая амплификация или сбой разъединения центриолей
        │
        └── Потеря дистальных придатков (CEP164, CEP89, Ninein, CEP170)
              → невозможность сборки первичной реснички
```

### 2.3 Два параллельных пути патологии

После накопления повреждений центриоль теряет две критические функции:

**Путь A — нарушение реснички (цилиарная слепота):**
```
Потеря CEP164/CEP89 → нет первичной реснички
→ стволовая клетка «глухая» к Shh и Wnt сигналам ниши
→ неверные решения о судьбе клетки
```

**Путь B — нарушение асимметричного деления (АКД):**
```
Деградация PCM → слабое и смещённое митотическое веретено
→ случайная сегрегация детерминантов судьбы (Numb, Prominin-1)
→ либо истощение пула (обе дочери дифференцируются)
→ либо гиперпролиферация (обе дочери остаются стволовыми)
```

### 2.4 Петля обратной связи (самоусиление)

```
Повреждённая центриоль
    │
    ▼
Дезорганизация микротрубочек
    │
    ▼
Нарушение митофагии → дисфункция митохондрий
    │
    ▼
↑ ROS → ещё больше повреждений центриоли (замкнутый круг)
```

Дополнительные петли:
- Смещение HDAC6/SIRT2 → глобальные эпигенетические изменения
- Ложный DDR-сигнал (BRCA1/53BP1 смещены) → p53/p21 → сенесценция

## 5. Модуль human_development_module

### `damage.rs` — Накопление повреждений

Функция `accumulate_damage()` реализует все пять типов повреждений за один
временной шаг `dt_years`. Ключевые особенности:

- **Возрастной множитель**: после 40 лет скорость повреждения умножается на
  `midlife_damage_multiplier` (антагонистическая плейотропия)
- **Петля ROS**: `ros_boost = 1 + α × total_damage_score()`
- **Потеря придатков необратима**: `cep164_integrity` только убывает, никогда
  не восстанавливается
- **Производные метрики** обновляются автоматически:
  ```
  ciliary_function  = mean(придатки) × (1 - агрегаты × 0.5)
  spindle_fidelity  = (1 - structure_damage) × (1 - phospho × 0.3)
  ```

### `inducers.rs` — Система S/H-индукторов

Трейт `InducerDivisionExt` для `CentriolarInducers`:

```rust
pub trait InducerDivisionExt {
    fn morphogenetic_level(&self) -> MorphogeneticLevel;
    fn asymmetric_divide(&mut self, spindle_ok: bool, rng_val: f32) -> DivisionOutcome;
}

pub enum DivisionOutcome {
    Asymmetric { stem_daughter, differentiating_daughter },
    SymmetricDifferentiation,   // истощение пула
    SymmetricSelfRenewal,       // клональная экспансия
    TerminalDifferentiation,    // S_count = 0
}
```

При `spindle_ok = false` (нарушенное веретено) деление с вероятностью 0.5
становится симметричным — либо истощение, либо экспансия.

### `tissues.rs` — Тканеспецифичные ниши

Каждая ткань имеет уникальный профиль (`TissueProfile`):

| Ткань | Множитель повреждений | Цилиарная чувствительность | Биологическое основание |
|-------|----------------------|---------------------------|-------------------------|
| `Hematopoietic` | ×1.3 | 0.9 | HSC: миелоидное смещение с возрастом |
| `Neural` | ×0.8 | 1.3 | NSC: критическая зависимость от Shh-реснички |
| `IntestinalCrypt` | ×1.2 | 0.7 | Высокий темп деления → быстрее истощается |
| `Muscle` | ×0.9 | 0.8 | Сателлиты: умеренная скорость деградации |
| `Skin` | ×1.1 | 0.6 | Эпителий: умеренные повреждения |
| `Germline` | ×0.5 | 1.0 | Половые клетки: защищены |

### `organism.rs` — Интеграция метрик

Маппинг тканевых метрик → глобальные показатели:

```
Neural.functional_capacity  →  cognitive_index
HSC.functional_capacity     →  immune_reserve
Muscle.functional_capacity  →  muscle_mass
(gut + HSC).senescent_fraction / 2  →  inflammaging_score
mean(все ткани).functional_capacity →  frailty_index
```

Смерть наступает при `frailty_index ≥ 0.95` или по достижении
`max_lifespan_years`.

## 6. Стадии развития организма

```
ЗИГОТА (день 0)
  Нет центриолей → тотипотентность
  S_count = 50, H_count = 4
  ROS минимальный
         │
         ▼ де-ново формирование центриолей
ДРОБЛЕНИЕ (1–4 дня)
  Центриоли только что сформированы — повреждений нет
  Делений: ~2/день
         │
         ▼
БЛАСТОЦИСТА (4–14 дней)
  ВКМ: плюрипотентные клетки, S_count ~40
  Трофобласт: начало дифференцировки
         │
         ▼
ГАСТРУЛЯЦИЯ → ОРГАНОГЕНЕЗ (2–8 недель)
  Три зародышевых листка, формирование органов
  Первые стволовые ниши, первый SASP-потенциал
         │
         ▼
ПЛОД (8 нед. – 9 мес.)
  Стремительное размножение стволовых клеток
  Первичные повреждения центриоли минимальны
         │
         ▼
ПОСТНАТАЛЬНЫЙ (0–18 лет)
  Активный рост → высокий темп деления
  Начало медленного накопления повреждений
         │
         ▼
ВЗРОСЛЫЙ (18–40 лет)
  Гомеостаз тканей, стабильная регенерация
  Повреждения нарастают линейно
         │
         ▼ [ключевой переломный момент по CDATA]
СРЕДНИЙ ВОЗРАСТ (40–65 лет)
  Антагонистическая плейотропия: ×1.6 скорость повреждений
  HSC: начало миелоидного смещения
  SASP активируется
  Immunosenescence, inflammaging
         │
         ▼
ПОЖИЛОЙ (65+ лет)
  Выраженная саркопения, иммунодефицит, когнитивный спад
  Пулы HSC, кишечника, кожи истощены
  Веретено нестабильно → клональная гематопоэтическая экспансия
         │
         ▼
СМЕРТЬ (~76–80 лет при норме)
  Frailty index ≥ 0.95
  Остаточная функция: Neural (когниция сохраняется дольше всего)
```

### Использование в коде

```rust
use human_development_module::{
    HumanDevelopmentModule, HumanDevelopmentParams,
    damage::DamageParams,
    development::DevelopmentParams,
};
use cell_dt_core::SimulationModule;

// Создать с параметрами по умолчанию
let mut module = HumanDevelopmentModule::new();

// Или с кастомными параметрами
let params = HumanDevelopmentParams {
    damage: DamageParams::longevity(),
    development: DevelopmentParams {
        s_inducers_initial: 50,
        max_lifespan_years: 120.0,
        ..Default::default()
    },
    steps_per_year: 10,
    seed: 42,
};
let mut module = HumanDevelopmentModule::with_params(params);

// Шаг симуляции (dt = 1 шаг = 0.1 года при steps_per_year=10)
let mut world = cell_dt_core::hecs::World::new();
module.initialize(&mut world).unwrap();
module.step(&mut world, 1.0).unwrap();

// Получить снимок метрик
let snap = module.snapshot();
println!("Возраст: {:.1} лет", snap.age_years);
println!("Дряхлость: {:.3}", snap.frailty);
println!("Когниция: {:.3}", snap.cognitive);
println!("Иммунитет: {:.3}", snap.immune);
```

## 9. Результаты и интерпретация

### Порядок истощения тканевых пулов

В нормальном режиме ткани деградируют в следующем порядке:

1. **Кожа** (`Skin`) — истощается первой (~50–55 лет)
2. **Кишечник** (`IntestinalCrypt`) — ~55–60 лет
3. **HSC** (`Hematopoietic`) — ~60–65 лет → иммунодефицит
4. **Мышцы** (`Muscle`) — ~70–75 лет → саркопения
5. **Нейральные СК** (`Neural`) — сохраняются дольше всего

Это соответствует клиническим наблюдениям: нейродегенерация — наиболее
поздний этап старения.

### Феномен: нейральные СК — последние защитники

Это не случайность: нейральные стволовые клетки имеют:
- Самый низкий множитель повреждений (×0.8)
- Наивысшую цилиарную чувствительность (×1.3) — они «слышат» Shh дольше
- Самый медленный темп деления (0.8/год) → медленнее истощаются

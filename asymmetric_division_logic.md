# Логика асимметричных делений: когда O₂ достигает центриолей

## Основной принцип

В норме митохондрии, расположенные преимущественно на периферии клетки, **поглощают кислород до того, как он достигает центра**. Центриолярная зона — самая кислородобеднённая область клетки (≈ микроаноксия).

Когда этот барьер нарушается, и O₂ / ROS проникают к центриолям — они наносят **асимметричный ущерб**: материнская центриоль (старше, уже имеет накопленные ПТМ) повреждается быстрее дочерней. Это нарушает баланс PLK4/SAS-6 и смещает ось веретена → **асимметричное деление**.

```
[O₂ у периферии] → [митохондрии поглощают] → [центриоли защищены]  → симметричное деление
[O₂ у периферии] → [барьер нарушен]         → [O₂ достигает центра] → асимметричное деление
```

---

## Сценарии, при которых O₂ достигает центриолей

### Сценарий 1: Переключение метаболизма (OXPHOS → гликолиз)
**Условие:** `warburg_index > 0.6` (клетка перешла на аэробный гликолиз)

Когда клетка переключается на гликолиз (как раковые клетки или эмбриональные SC), митохондрии снижают потребление O₂. Кислород больше не поглощается у периферии и диффундирует к центру.

- **Когда:** Активация онкогенов (MYC, HIF-1α), трансформация, ранний эмбриогенез
- **Биомаркер:** lactate↑, O₂ consumption rate↓, цитоплазматическое O₂↑
- **Порог в модели:** `mitochondrial_activity < 0.4` AND `ros_level > 0.2`
- **Исход:** Мягкое асимметричное деление; дочерняя клетка с материнской центриолью чуть более дифференцирована

---

### Сценарий 2: Митохондриальная асимметрия перед делением
**Условие:** Митохондрии кластеризуются к одному полюсу клетки

Перед митозом митохондрии активно перераспределяются. Если их больше у апикального полюса → у базального полюса O₂ не поглощается → O₂ достигает центриоли у базального полюса.

- **Когда:** Делящиеся нейральные стволовые клетки (apical/basal полярность), эпителиальные SC
- **Биомаркер:** Асимметричный митохондриальный потенциал (ΔΨm), асимметрия TFAM
- **Порог в модели:** `mitochondrial_polarity_index > 0.5` (отношение апикальный/базальный пул)
- **Исход:** Базальная дочерняя клетка → более дифференцированная; апикальная → SC

---

### Сценарий 3: Гипоксия-реоксигенация (ишемия-реперфузия)
**Условие:** `oxygen_tension` резко возрастает после периода гипоксии

После периода низкого O₂ митохондрии переходят в режим "slow burn". При внезапной реоксигенации поток O₂ превышает митохондриальную ёмкость → ROS-взрыв → O₂ и H₂O₂ проникают к центру.

- **Когда:** Мышечные SC (post-exercise), гемопоэтические SC после инфаркта, ишемия кишечника
- **Биомаркер:** Burst ROS, карбонилирование центриолярных белков SAS-6/CEP135
- **Порог в модели:** `ros_spike_magnitude > 0.5` за `dt < 2` дня после `oxygen_tension < 0.2`
- **Исход:** Форсированное асимметричное деление, возможно с апоптозом одной дочерней клетки

---

### Сценарий 4: Истощение антиоксидантного барьера (старение)
**Условие:** `age_years > 40` AND `total_damage_score > 0.15`

С возрастом снижается экспрессия SOD2, каталазы, глутатиона (GSH), SIRT3. Нейтрализация ROS у периферии падает — O₂ и H₂O₂ проникают глубже в цитоплазму, доходя до центриолей.

- **Когда:** Возраст-ассоциированное истощение SC (HSC, кишечные SC после 50 лет)
- **Биомаркер:** GSH↓, 8-OHdG↑, карбонилирование центриолярных белков↑
- **Порог в модели:** `antioxidant_capacity = 1.0 - damage.protein_carbonylation` < 0.5
- **Исход:** Постепенный сдвиг от симметричного самообновления → асимметричному → дифференцировке
- **Связь с CDATA:** Это главный долгосрочный механизм; напрямую связан с `midlife_damage_multiplier × 1.6`

---

### Сценарий 5: SASP-индуцированное паракринное повреждение
**Условие:** `senescent_neighbor_fraction > 0.25`

Сенесцентные клетки secretируют H₂O₂, IL-6, MMP. H₂O₂ диффундирует через мембрану соседней SC, проходит цитоплазму и достигает центриолей — особенно если внутриклеточный барьер уже ослаблен.

- **Когда:** Стареющие ниши (кроветворный костный мозг, стареющая крипта кишечника)
- **Биомаркер:** pH2AX в соседних клетках, CEP164↓ у клеток рядом с сенесцентными
- **Порог в модели:** `tissue_state.senescent_fraction > 0.3` AND `cep164_integrity < 0.7`
- **Исход:** Принудительное асимметричное деление; снижение S-индукторов быстрее нормы

---

### Сценарий 6: Отмена сигналов ниши (нишевый коллапс)
**Условие:** `niche_signal_strength < 0.3` (снижение Wnt, Notch, SCF)

Сигналы ниши (Wnt, SCF, Notch) поддерживают метаболически активное состояние SC с плотным митохондриальным щитом. При снижении сигналов клетка входит в глубокую покой (G0) → метаболизм падает → O₂ не поглощается.

- **Когда:** Посттравматическое состояние, воспаление, фиброз ниши
- **Биомаркер:** LKB1↑, AMPK↑, mTOR↓, уровень ATP↓
- **Порог в модели:** `ciliary_function < 0.5` (цилии — сенсоры Wnt/Shh) AND `regeneration_tempo < 0.3`
- **Исход:** Нарушение баланса самообновления; ниша "истощается" быстрее расчётного

---

### Сценарий 7: Крупноклеточный порог (геометрия диффузии)
**Условие:** Клетка физически большая (оocит, некоторые эмбриональные клетки)

O₂ диффундирует со скоростью ~200 мкм²/с. В клетке диаметром > 100 мкм митохондрии у периферии не успевают поглотить весь поток — O₂ диффундирует к центру пассивно.

- **Когда:** Оогенез (ооцит ~120 мкм), бластомеры ранней дробления, гигантские клетки Пуркинье
- **Биомаркер:** Градиент [O₂] по диаметру клетки
- **Порог в модели:** Специфично для `DevelopmentalStage::Cleavage` и `Zygote`
- **Исход:** Деление De novo (у зиготы центриоли образуются заново — особый случай без материнской центриоли)

---

### Сценарий 8: Геnotoxic стресс через p53
**Условие:** `dna_damage_level > 0.4`

Активация p53 при повреждении ДНК форсирует метаболический сдвиг: клетка снижает OXPHOS (через TIGAR, SCO2), митохондриальная активность падает. O₂ проникает глубже. Одновременно p53 стабилизирует PLK4 (регулятор центриоль → деление), меняя баланс.

- **Когда:** Геnotoxic стресс (UV, ионизирующее излучение, химиотерапия)
- **Биомаркер:** p21↑, γH2AX↑, PLK4 изменение локализации
- **Порог в модели:** Из `CellCycleStateExtended::growth_factors.dna_damage > 0.4`
- **Исход:** Чаще всего → арест цикла, но при субпороговых уровнях → асимметричное деление с сохранением SC

## Механизм асимметрии материнской/дочерней центриоли

Когда O₂ достигает центриолей, повреждение **не одинаково** для материнской и дочерней:

```
Материнская центриоль:
  - Старше (накоплено больше ПТМ: hyperacetyl, carbonyl)
  - Дистальные придатки (CEP164, CEP89) уже частично повреждены
  - Выше локальная концентрация агрегатов CPAP/CEP290
  → Более уязвима к O₂/ROS
  → Получает бо́льшее повреждение за тот же dt

Дочерняя центриоль:
  - Молодая (pristine или слабо повреждённая)
  - PLK4 активен → реагирует на ROS, но быстро восстанавливается
  → Менее уязвима

Результат:
  damage(мать) >> damage(дочь)
  → spindle_fidelity[мать] < spindle_fidelity[дочь]
  → Ось веретена смещается
  → Детерминанты судьбы (Numb, Notch, Musashi) сегрегируют неравномерно
  → Дочерняя клетка с материнской центриолью: дифференцировка
  → Дочерняя клетка с дочерней центриолью: самообновление
```

## Предлагаемые параметры для AsymmetricDivisionComponent

```rust
pub struct AsymmetricDivisionTrigger {
    /// Локальная [O₂] у центриоли (0.0 = аноксия, 1.0 = гипероксия)
    pub centrosomal_oxygen_level: f32,

    /// Асимметрия повреждения мать/дочь (0 = симметрично, 1 = максимальная разница)
    pub mother_daughter_damage_delta: f32,

    /// Активен ли O₂-зависимый триггер асимметрии
    pub oxygen_trigger_active: bool,

    /// Тип активного сценария (для логирования)
    pub active_scenario: OxygenScenario,
}

pub enum OxygenScenario {
    None,
    WarburgSwitch,       // Сценарий 1
    MitochondrialPolar,  // Сценарий 2
    HypoxiaReoxyg,       // Сценарий 3
    AntioxidantDepletion,// Сценарий 4
    SaspParacrine,       // Сценарий 5
    NicheCollapse,       // Сценарий 6
    GeometricDiffusion,  // Сценарий 7
    P53Activation,       // Сценарий 8
}
```

## Логика `step()` для AsymmetricDivisionModule

```
fn step():
  для каждой сущности с (CentriolarDamageState, TissueState, CellCycleStateExtended):

    1. Вычислить centrosomal_oxygen_level:
       = 1.0 - mitochondrial_shield(damage, tissue)

    2. Проверить сценарии (по приоритету):
       if ros_level > 0.5 AND кратковременный → HypoxiaReoxyg
       elif mito_activity < 0.4 → WarburgSwitch
       elif senescent_fraction > 0.3 → SaspParacrine
       elif antioxidant_capacity < 0.5 → AntioxidantDepletion
       elif niche_signal < 0.3 → NicheCollapse
       ...

    3. Если centrosomal_oxygen_level > порог:
       mother_daughter_delta = f(protein_carbonylation, acetylation, age)
       if delta > 0.15 → DivisionType = Asymmetric
       if delta > 0.5  → DivisionType = Differentiation

    4. Списать S-индуктор при каждом асимметричном делении
```

## Вывод

Ключевой предсказательный вывод CDATA для асимметричных делений:

> **Асимметричное деление — не случайное событие, а детерминированный ответ на кислородный доступ к центриолям, нарушающий баланс мать/дочь.**

Старение SC-пула происходит именно потому, что с возрастом всё больше делений становятся асимметричными (через Сценарий 4 + 5), постепенно смещая баланс в сторону дифференцировки и истощая резерв стволовых клеток.
